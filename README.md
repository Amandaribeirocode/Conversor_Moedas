💱 Conversor de Moedas

Aplicação web para conversão de moedas com interface moderna e fácil de usar, permitindo ao usuário obter valores convertidos de forma rápida e prática.

📌 Sobre o projeto

Este projeto foi desenvolvido como parte da minha jornada de aprendizado em programação, com foco em desenvolvimento front-end.

A aplicação permite que o usuário insira um valor, selecione as moedas de origem e destino e visualize o valor convertido instantaneamente.

✨ Funcionalidades
Conversão de moedas em tempo real 💰
Seleção de moeda de origem e destino
Interface intuitiva e responsiva
Exibição clara do valor convertido
🎨 Interface

Interface clean, moderna e centrada na experiência do usuário.
